design system

a coherent visual paradigm for the 247420 collective. layered surfaces, monospace labels, loud content inside quiet chrome.

stack

webjsx.org + customized ripple-ui.com on top of tailwind. buildless in the browser, vendored not cdn'd. build box is github actions (preferred for everything — build, test, deploy). cms surfaces use flatspace as a gh actions step (never bunx / npx). non-cms surfaces that need data use busybase from npm (not supabase, not firebase).
